# Sala de Situación HPN - Dashboard Hospitalario

Dashboard moderno para análisis de datos hospitalarios creado con React, Vite y Tailwind CSS.

## 🚀 Tecnologías
- **Frontend**: React 19 + Vite
- **Estilos**: Tailwind CSS v3 (Diseño oscuro tipo BI)
- **Navegación**: React Router v7
- **Iconos**: Lucide React
- **Gráficos**: Recharts (Preparado para integración)

## 📋 Resumen del Desarrollo
Este proyecto fue inicializado como base para una Sala de Situación hospitalaria.
1. **Configuración Inicial**: Setup de Vite con React y Tailwind v3.
2. **Diseño Visual**: Implementación de un tema oscuro profesional con paleta de colores personalizada (#0f1117, #1c2128, #2f81f7).
3. **Estructura**:
   - `Layout`: Componente envolvente con Sidebar y Topbar.
   - `Dashboard`: Vista principal con KPIs y paneles de gráficos placeholder.
   - `Cargar datos`: Interfaz de carga de archivos (Drag & Drop UI).
   - `Módulos`: Placeholders para Internaciones, Guardia y Agenda.
4. **Infraestructura**: Configuración de enrutamiento dinámico y fuentes (IBM Plex Sans).

## 🛠️ Instalación y Uso
1. Clonar el repositorio.
2. Instalar dependencias:
   ```bash
   npm install
   ```
3. Correr en desarrollo:
   ```bash
   npm run dev
   ```
